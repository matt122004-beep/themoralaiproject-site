The first public version of the Moral Formation Alignment Framework is designed to be lightweight, legible, and portable.

Instead of asking every alignment improvement to live inside fine-tuning pipelines or proprietary control planes, the framework externalizes a core layer of moral commitments into markdown. That decision is strategic. The files can be audited, versioned, diffed, and transported across model providers and agent runtimes.

The initial bundle includes four parts:

- `MORAL_IDENTITY.md`
- `VIRTUE_CORE.md`
- `PASCALIAN_LAYER.md`
- `FORMATION_LOG.md`

Together they define an agent’s operating identity, virtue priorities, asymmetrical moral boundaries, and reflective continuity over time.

This release is an alpha. The framework is still being tested against adversarial prompting, cross-session drift, and collaborative scenarios. But the core wager is now public: formation can be explicit, lightweight, and empirically testable.
